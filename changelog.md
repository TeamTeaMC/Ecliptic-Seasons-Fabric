This update mainly contains minor adjustments and improvements.
For more detailed information, please refer to the changelog for `0.13.0-rc` and `0.13.0-rc-2`.

#### Improvements

* Increased the default value of `fixed_seed_chance` when `fixed_seed` is enabled in the season definition datapack
* When adjusting items that require both a world restart and a game restart, the game will now prompt the player to restart the game
* Fixed an issue where the configuration screen could not be opened in multiplayer environments (1.20.1 only). Note that editing server-side configurations from the client is not supported in any version
* Fixed an issue where seasonal ambient sounds would not stop under certain abnormal conditions
